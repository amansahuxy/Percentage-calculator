# 📊 Grade Calculator

A simple Grade Calculator built using HTML, CSS, and JavaScript. Users can enter obtained marks and total marks to calculate their percentage and grade instantly.

## 🚀 Features

* Calculate percentage from marks
* Automatic grade evaluation
* Displays percentage result
* Displays grade instantly
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* JavaScript

## 📁 Project Structure

```text
Grade-Calculator/
│
├── index.html
├── index.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## 🧠 How It Works

1. Enter obtained marks.
2. Enter total marks.
3. Click the **Calculate** button.
4. The app calculates the percentage.
5. The corresponding grade is displayed.

## 🎓 Grading System

| Percentage | Grade |
| ---------- | ----- |
| 90 - 100   | A+    |
| 80 - 89    | A     |
| 70 - 79    | B+    |
| 60 - 69    | B     |
| 50 - 59    | C+    |
| 40 - 49    | C     |
| Below 40   | Fail  |
